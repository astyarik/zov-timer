# ZOV Таймер, это дискорд бот который:

Отправляет одно сообщение и каждую секунду увеличивает количество ZOV
Показывает сколько это в секундах
Работает через .zov и слэш-команду /zov

## Функционал:
Обычная команда
.zov [секунд]
(количество секунд таймера (по умолчанию 60))
Максимум 100 секунд

Кулдаун 10 секунд на пользователя

Пример использования:

.zov # 60 секунд
.zov 30 # 30 секунд

Бот будет редактировать одно сообщение каждую секунду, добавляя ZOV

Слэш-команда

/zov seconds:<число>

Тоже количество секунд таймера (по умолчанию 60, максимум 100)

Пример использования слэш-команд:

/zov
/zov seconds:30

Функция полностью аналогична prefix-команде

Требования:

Python 3.10+
discord.py 2.x
python-dotenv

Установка зависимостей:
`pip install -r requirements.txt`

Переменное окружение:
Создайте .env файл в корне проекта:
`DISCORD_TOKEN=ВАШ_ТОКЕН_БОТА`

Запуск
`python bot.py`

Удачи в ПодZчете Vремени!
